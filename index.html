Project Description

Abstract

What this project is, why it matters, and what we built.

Skin diseases affect nearly 900 million people globally, yet qualified dermatological expertise remains severely limited in developing countries such as Pakistan — a nation of 230 million with fewer than 500 registered dermatologists. This Final Year Project presents the design, development, and deployment of a fully embedded, real-time AI system for skin lesion classification, implemented on the ESP32-CAM microcontroller — requiring no cloud connectivity and costing under PKR 1,500.

The system leverages a curated subset of the HAM10000 dermoscopic dataset (~2,800 images), training a FOMO (Faster Objects, More Objects) neural network with MobileNetV2 backbone and Int8 quantisation via the Edge Impulse platform. The deployed model classifies lesions as either Melanocytic Nevi (NEVUS) or Basal Cell Carcinoma (BCC) with over 83% accuracy at under 300 ms per inference, entirely on-device.

A comprehensive browser-based web dashboard is served directly from the ESP32 over local Wi-Fi, displaying a live camera preview, a gallery of captured images labelled by disease name, stats and CSV export. The system runs on a standard USB power bank, and is deployable by community health workers in the most resource-constrained environments.

Technologies

Tools & Technologies

The complete tech stack powering this embedded AI system from data to deployment.

🤖
ML Platform
Edge Impulse
Cloud ML platform for training, optimising, and exporting FOMO models as Arduino libraries for embedded deployment.
📷
Hardware
ESP32-CAM
Dual-core 240 MHz SoC with 4 MB PSRAM, OV2640 2 MP camera, and built-in Wi-Fi. Entire inference runs on this chip.
⚡
IDE
Arduino IDE
Firmware development, compilation, and UART upload for ESP32-CAM with full OV2640 camera and Wi-Fi stack control.
🧬
Dataset
HAM10000
10,015 dermoscopic images across 7 classes. Project uses ~2,800 images (NV + BCC) validated by histopathology.
🧠
Model
MobileNetV2 / FOMO
Lightweight CNN with depthwise separable convolutions. Int8 quantised to 75 KB — fits ESP32 memory budget.
🌐
Web
HTML / CSS / JS
Single-page dashboard served from ESP32 HTTP server. Live camera, image gallery, stats and CSV export.
🐍
Data Science
Python + Pandas
Dataset preprocessing, EDA, class distribution visualisation, image resizing pipeline, and train/test split.
🔢
Runtime
TF Lite Micro
TensorFlow Lite Micro runtime inside the Edge Impulse Arduino library — neural network inference directly on ESP32.
🐙
Version Control
GitHub
Full source code, firmware, preprocessing scripts, and documentation hosted publicly for reproducibility.
